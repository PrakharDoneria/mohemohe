## Getting Started

### Prerequisites

- Node.js

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/princekunal101/mohemohe.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Configure environment variables
   ```sh
   cp .env.example .env
   ```
