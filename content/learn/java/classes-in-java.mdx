## Data Types
Data types specify the type of data that can be stored in a variable. Java supports several built-in data types, which can be categorised into two main categories &mdash;
- Primitive data types and
- Refrence data types

### Primitive data types
```java title="Example.java"
int myNum = 7;                // Integer
float myFloatNum = 8.74f;     // Floating Point Number
char myChar = 'P';            // Character
boolean myBool = true;        // Boolean
String myText = "Hello";      // String
```
There are mainly eight types of primitive data types in Java programming &mdash;
| Data Type | Characteristics | Size | Range |
|:---------:|:--------------- |:---- |:-----|
| `byte` | **8 bit** signed integer | **1 byte** | **-128** to **127** |
| `short` | **16 bit** signed integer | **2 bytes** | **-32768** to **32767** |
| `int` | **32 bit** signed integer | **4 bytes** | **2E-31** to **2E+30** |
| `long` | **64 bit** signed integer | **8 bytes** | **2E-63** to **2E+62** |
| `float` | **32 bit** floating point number | **4 bytes** | **&pm; 1.4E-45** to **&pm; 3.4028235E+38** |
| `double` | **64 bit** floating point number | **8 bytes** | **&pm; 4.9E-324** to **&pm; 1.7976931348623157E+308** |
| `boolean` | `true` or `false` | **1 bit** | **NA** |
| `char` | **16 bit**, Unicode | **2 bytes** | **Unicode character**, `\u0000` to `\uFFFF` can mix with hexadecimal number. |
