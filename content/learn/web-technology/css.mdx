## CSS
- Cascading Style Sheet language used for describing the presentation of the document written in Markup language such as HTML.
- CSS is a corner store technology of the world wide web along side HTML and JavaScript.
- CSS is easy to learn and understand but it provides powerful control over the presentation of the HTML document.

## Why CSS
- **CSS saves time:** You can write CSS once and reuse same seat in multiple HTML pages.
- **Easy maintenance:** To make a global changes, Simply change the style and all the elements in all the webpages will be updated automatically.
- **Search engines:** CSS is considered as a clean coding technique which means search engine could not have a struggle to reach its content.
- **Superior style to HTML:** CSS has a much wider array of attribute than HTML so you can give a far better look at your HTML pages in comparison to HTML attributes.

### CSS Syntax
- A rule consist of a selector & decleration blocks.
- The selector pointer to the HTML element you want to style.

<FigureMdx lightUrl="wt-css-syntax" desc="CSS Syntax diagram"/>
- The decleration pop contains one or more decleration saperated by semicolons.
- Each decleration includes are CSS property name and a value saperated by a colon ``` : ```.
- Multiple CSS decleration are saperated with semicolons and decleration are surrounded by curly bracess ```{ }```.

```css title="example.css"
p {
    color: green;
    text-align: center;
}
```
- Where ```p``` is selector ```color``` is property ```green``` is property value & ```text-align``` is property and ```center``` is value.

## CSS Selector
CSS selector are used to find or select HTML element based on their element name, id, class attribute and more. Some selectors are listed below &mdash;
- **The Universal Selector**: Rather then selecting elements of a specific type the universal selector quite simply matches the name of any element type.

```css
* {color: green;}
```

- **The Element Selector**: The element selecotr selects elements based on the element name. You can select all ```p``` element on a page like this.
```css
p {text-align: center; color: blue;}
```

- **The Descendant Selector**: It is used when developer wants to apply a style rule to a particular elements only when it lies inside a particular elements.
```css
ol p {color: green;}
```

- **Class Selector**: It can be used to identify more than one element. In CSS, a class selector is a name proceded by ```.``` full stop.
```css
. class_selector_name { property: value; }
```

- **Id Selector**: It can be used identify one element. In CSS and id selector is a name preseded by ```#``` hash selector.
```css
# id_selector_name {property: value;}
```
